GoChat - Real-time Chatting Application

Brief Description

A full-stack web application designed for instant messaging and real-time collaboration.

Tech Stack

  • Backend: Spring Boot (Java)
  • Frontend: ReactJS (Axios, StompJS, SockJS-client)
  • Real-time Engine: WebSocket (STOMP Broker)
  • Database: MySQL

System Architecture

  • RESTful API for historical message queries and authentication handling
  • WebSocket Connection Upgrade for persistent full-duplex client-server pipeline
  • Chat / Broker Service to:
    • Parse real-time network payloads dynamically
    • Dispatch text packets safely to specific active conversation nodes
    • Broadcast online states and typing triggers instantly

REST API Gateways

Method Endpoint Access Description
POST /login Public Processes login claims and issues JWT tokens.
GET /my-conversations Private Loads chat histories with last-message optimization models.
GET /chat-history Private Loads legacy messages for selected rooms.

WebSocket Channels

  • Handshake URL: http://localhost:8080/ws-chat
  • Inbound Topic Feed: /topic/room/{conversationId}
  • Outbound Publish Route: /app/chat/{conversationId}

🚀 Usage & Local Installation Guide

Follow these quick sequential steps to bootstrap and set up the project locally:

📦 Click to expand installation instructions

1. Set Environment Variables

Before launching the application server, export the required hidden key secrets in your operating system terminal window:

# On Linux / macOS systems
export DB_USERNAME="your_database_user"
export DB_PASSWORD="your_secure_password"
export JWT_SECRET_KEY="SuperSecretRandomKeyGeneratedForProductionUseMustBeLong"

# On Windows systems (Command Prompt)
set DB_USERNAME=your_database_user
set DB_PASSWORD=your_secure_password
set JWT_SECRET_KEY=SuperSecretRandomKeyGeneratedForProductionUseMustBeLong

2. Compile and Start Backend Server

Use Maven wrapper tool to compile source code assets and host the embedded server container:

mvn clean install
mvn spring-boot:run

The core REST services engine will spin up on default local connection host endpoint: http://localhost:8080
